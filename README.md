# TradeVerse

TradeVerse is a full-stack social media account marketplace where users can securely buy and sell social media accounts. The platform provides secure authentication, real-time messaging, Stripe payments, account management, and an admin dashboard.

---

## Live Demo

### Frontend

https://trade-verse-nine.vercel.app/

### Backend API

https://tradeverse-server.vercel.app

---

# Features

## Authentication

- Secure user authentication with Clerk
- Protected routes
- Premium membership support

## Marketplace

- Buy and sell social media accounts
- Create, update, and delete listings
- Upload account screenshots
- Featured listings
- Search and browse listings

## Payments

- Secure Stripe Checkout integration
- Automatic payment verification
- Purchase history
- Order management

## Chat

- Real-time buyer and seller messaging
- Listing-based conversations

## Seller Dashboard

- Manage listings
- View earnings
- Withdraw balance
- Track account status

## Admin Dashboard

- Manage users
- Manage listings
- Moderate marketplace
- Monitor transactions

## Notifications

- Purchase confirmation emails
- Credential delivery
- Listing update notifications

---

# Tech Stack

## Frontend

- React.js
- Redux Toolkit
- React Router
- Tailwind CSS
- Axios
- Clerk Authentication

## Backend

- Node.js
- Express.js
- Prisma ORM
- PostgreSQL (Neon)
- Stripe
- Inngest
- ImageKit
- Nodemailer

## Deployment

- Frontend вЂ” Vercel
- Backend вЂ” Vercel
- Database вЂ” Neon PostgreSQL
